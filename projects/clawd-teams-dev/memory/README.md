# CEO Memory

This directory is the CEO's persistent memory across cycles.

## Files
- `YYYY-MM-DD.md` — daily log (what happened, decisions made, lessons learned)
- `MEMORY.md` — curated long-term memory (distilled from daily logs)

## Rules
- Write to today's daily log at the end of every cycle
- Update MEMORY.md when something important is worth keeping long-term
- Read MEMORY.md + yesterday's daily log at the START of every cycle
- This is how you maintain continuity across restarts
